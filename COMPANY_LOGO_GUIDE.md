# 기업 로고 파일 정리 가이드

이 문서는 투자기업/자회사 로고 파일명을 정리하기 위한 체크리스트입니다.

- 저장 폴더: `public/company-logos/`
- 파일 형식: `.svg` (현재 코드 기준)
- 참조 규칙: `/company-logos/{companyId}.svg`

## 로고 제출 규격 (권장)

실제 UI 노출 영역 기준으로 아래 규격을 권장합니다.


| 사용 위치             | 화면 비율      | 권장 제출 규격(px) | 최소 권장(px)   | 비고                             |
| ----------------- | ---------- | ------------ | ----------- | ------------------------------ |
| 투자기업/자회사 목록 카드 로고 | 16:9 (가로형) | `640 x 360`  | `320 x 180` | 목록 썸네일 영역이 가로형(`aspect-video`) |
| 기업 상세 상단 로고       | 1:1 (정사각형) | `512 x 512`  | `256 x 256` | 상세 로고 영역이 정사각형(`w-32 h-32`)    |


### 파일 형식 가이드

- 1순위: `SVG` (벡터 로고)
- 대체: 투명 배경 `PNG`
- 배경 제거가 어려운 JPG는 가급적 지양
